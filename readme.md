## Getting Started

To get a local copy up and running follow these simple steps.


### Installation
 
1. Clone the repo
```sh
git clone https://github.com/felipealvescosta/upload-file-backend.git
```
Run Project
<br/>
1. Enter the server folder
```sh
cd upload-file-backend
yarn install 
yarn dev
```
